# AI Background Removal Web Application 🖼️✨

A modern, fast, and responsive web application that automatically removes backgrounds from images using Python/Flask and the Gemini API.

---

## 🚀 Features

* **Instant Background Removal:** Leverages the power of AI to cleanly isolate subjects from their backgrounds.
* **Modern UI/UX:** Styled with contemporary design trends, including dark mode accents, smooth keyframe animations, and clean layouts.
* **Real-time Preview:** Upload and view your processed images directly through an interactive HTML/CSS frontend.
* **Flask Backend:** A lightweight, optimized Python server managing API requests and image processing pipelines securely.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Modern Animations & Custom Styling), Vanilla JavaScript
* **Backend:** Python, Flask
* **AI Integration:** Gemini API
* **Image Utilities:** Pillow / OpenCV
